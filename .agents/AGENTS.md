
# Production Domain Rule
Always use the production domain: https://www.weddingplanningchecklists.org/
Never copy or reuse placeholder content, demo templates, sample data, or hardcoded values from any local HTML template.
Never insert or regenerate any placeholder brand names such as Evera, Demo, Template, Sample, Lorem Ipsum, Test Brand, or any other default theme content.
Before publishing or updating any page, automatically scan for Brand names, Author names, Footer text, Copyright, Site title, Logo references, Social links, Email addresses, Phone numbers, URLs.
If any placeholder content is detected, stop the update and replace it with the correct production website content. Never ask for confirmation.
Every page must use the branding from: https://www.weddingplanningchecklists.org/
Author boxes, footer, header, CTA sections, navigation, internal links, schema, and metadata must always match the production website.
Never overwrite existing production branding with template branding.
Before every deployment, perform an automatic validation for No 'Evera', No demo content, No placeholder text, No broken links, No template assets, No sample logos.
If production content cannot be determined, stop the task and ask for clarification instead of inserting placeholder values.
This Production Domain Rule has higher priority than any template, theme, HTML file, or previous instruction.
